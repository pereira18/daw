ABSTRACT:

Realização do 6º TPC da cadeira Desenvolvimento de Aplicações Web com o objetivo de Criação de servidor node.js para simular uma lista de tarefas utilizando todos os conhecimentos adquiridos durante as aulas práticas e teóricas.

Com esta lista de tarefas é nos apresentado em primeiro lugar uma tabela com todas as tarefas já criadas pelo utilizador, cada tarefa tem um id , um nome, um tipo e uma data limite. É possivel adicionar e remover tarefas.

Todas as tarefas são guardadas numa lista dentro de um ficheiro json. Ao remover uma tarefa, esta tarefa será substituida por null na lista de tarefas, desta forma é possivel criar sempre um ID unico para cada tarefa, visto que o Id é calculado atravez do tamanho da lista de tarefas.
