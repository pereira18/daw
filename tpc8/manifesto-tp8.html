ABSTRACT:

Realização do 8º TPC da cadeira Desenvolvimento de Aplicações Web com o objetivo de Criação de servidor node.js para simular o upload de vários ficheiros utilizando todos os conhecimentos adquiridos durante as aulas práticas e teóricas.

É nos apresentado em primeiro lugar 2 campos, um para adicionarmos um ficheiro e o outro para adicionarmos a descrição para o respetivo ficheiro. Mais abaixo é nos apresentado uma tabela com todos os ficheiros já submetidos, apresentando o nome descrição e data de submissão de cada ficheiro.

Todas os ficheiros foram guardados numa lista dentro de um ficheiro json. Ao contrario do ultimo TPC todas as tarefas são realizadas na mesma página e a página é atualizada parcialmente, pois foi utilizado ajax.
