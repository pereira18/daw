ABSTRACT:

Realização do Quinto TPC da cadeira Desenvolvimento de Aplicações Web com o objetivo de Criação de servidor node.js para a visualização de obras musicais utilizando todos os conhecimentos adquiridos durante as aulas práticas e teóricas.

Para a realização deste TPC foi necessário criar um script para a criação de um ficheiro index.json capaz de apresentar um indice para todos os ficheiros json existentes, foi corrigido um erro num ficheiro json o que tornava impossivel a leitura do mesmo.

O trabalho é composto por uma página inicial onde as obras estão separadas por tipo. Atravéz deste indice é possivel aceder ao conteudo de todas as obras.
